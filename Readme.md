# WGS84-CGCS2000 坐标转换工具

一个高效、准确的WGS84经纬度坐标与国家2000高斯投影坐标双向转换工具，支持单点和批量转换功能。

## 功能特性

### 🔄 双向坐标转换
- **WGS84经纬度 → 国家2000高斯投影坐标**
- **国家2000高斯投影坐标 → WGS84经纬度**

### 📐 灵活的转换配置
- 支持3°分带和6°分带
- 可选择是否启用带号
- 内置权威EPSG编码配置

### 📊 批量处理能力
- 支持Excel(.xlsx)、CSV(.csv)、TXT(.txt)文件导入导出
- 自动验证数据格式，提供详细错误信息
- 批量结果表格预览（最多50条）

### 🌐 用户友好界面
- 基于Bootstrap 5的响应式设计
- 实时操作日志记录
- 简洁直观的交互流程

## 技术栈

- **后端**: Python 3.12, Flask 3.1.2
- **坐标转换**: pyproj 3.7.2
- **数据处理**: pandas 2.3.3, openpyxl 3.1.5
- **Web服务器**: Gunicorn 23.0.0
- **容器化**: Docker
- **CI/CD**: GitHub Actions

## 快速开始

### 使用Docker运行

```bash
# 克隆项目
git clone <repository-url>
cd WGS84-CGCS2000

# 构建并运行Docker容器
docker compose up -d
```

服务将在 http://localhost:5000 上运行

### 本地开发环境

```bash
# 安装依赖
pip install -r requirements.txt

# 运行开发服务器
python main.py
```

## 使用说明

### 单点转换
1. 在"单点转换"标签页中选择转换方向
2. 设置分带类型（3°或6°）和带号选项
3. 输入相应的坐标值
4. 点击"执行转换"查看结果

### 批量转换
1. 在"批量转换"标签页中选择转换方向
2. 点击"导出参考模板"下载相应格式的模板文件
3. 按照模板格式填写数据并保存
4. 上传数据文件并点击"开始批量转换"
5. 转换完成后可预览结果并点击"导出转换结果"获取完整数据

## 坐标系统说明

- **WGS84**: 全球定位系统(GPS)使用的坐标系统，以经纬度表示
- **国家2000(CGCS2000)**: 中国新一代大地坐标系，高斯投影以米为单位
- **分带规则**: 
  - 3°分带范围：25-45带
  - 6°分带范围：13-23带

## 精度与限制

- WGS84经纬度范围：经度72-135°，纬度0-54°
- 坐标转换保留3-6位小数，确保高精度
- 单个文件处理建议不超过10万条记录
